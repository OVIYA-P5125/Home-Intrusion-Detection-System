# Home Intrusion Detection System

## Overview
The Home Intrusion Detection System is an Arduino Uno-based security project that detects unauthorized entry using RFID authentication, a PIR motion sensor, and a magnetic door sensor. The system triggers a buzzer and LED alarm when intrusion is detected while the system is armed.

## Features
- RFID-based system arm/disarm
- Motion detection using PIR sensor
- Door open detection using magnetic sensor
- Buzzer and LED alarm system
- Unauthorized access detection

## Components Used
- Arduino Uno
- RC522 RFID Module
- PIR Motion Sensor
- Magnetic Door Sensor
- Buzzer
- LED
- Breadboard and Jumper wires

## Working
1. Scan authorized RFID card to arm/disarm system
2. When armed:
   - Motion detection triggers alarm
   - Door opening triggers alarm
3. Unauthorized RFID triggers buzzer alert
